# scip_ahead
A lightweight, open source code intelligence tool

SCIP, Standard Code Intelligence Protocol, is a serialization format that describes information about computer source code.  It makes details about source documents, symbols, and relationships available for use by processes that operate on that source code. 

Many Large Language Model coding tools are still using raw text search with no deeper understanding of the underlying structures in the code, leading to poor decision making, bugs and hallucinations. SCIP Ahead is an open source MCP tool that gives AI agents the capability to generate, update and search SCIP data across multiple repositories.

Currently the only supported language is .Net.  Javascript is the next language scheduled for implementation, followed by Python.
